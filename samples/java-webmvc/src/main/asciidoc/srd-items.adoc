== SARD/SRD mix

This part of the documentation demonstrates the combination of Spring Auto REST Docs with Spring REST Docs.
The path parameters and response fields are manually documented with Spring REST Docs.
The authorization and other snippets are still coming from Spring Auto REST Docs.
The cURL and HTTP snippets are coming from Spring REST Docs in any case.
All is configured together to compose a convenient section snippet.
